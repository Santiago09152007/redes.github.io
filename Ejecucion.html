Fase Ejecución

Avance Tecnico - Español

En esta fase del proyecto se evidencia como se destaca la presencia de un banco de pruebas de 100 cm x 100 cm. Este banco cuenta con la instalación de tres puntos de red, tres puntos eléctricos normales y tres puntos eléctricos regulados. Además, se proporciona una simulación lógica de la red que muestra cómo funciona y cómo está configurada la red inalámbrica, de acuerdo con los parámetros solicitados. Como prueba visual, se incluyen videos de la implementación, planos en 3D y facturas de compra. También se implementa el software SpiceWorks para la gestión del banco de pruebas y su creación.

Avance Tecnico - Inglés

In this phase of the project, the presence of an 100 cm x 100 cm test bench is evident. This bank has the installation of three network points, three normal electrical points and three regulated electrical points. Additionally, a logical network simulation is provided that shows how the wireless network works and is configured, according to the requested parameters. As visual proof, implementation videos, 3D plans and purchase invoices are included. SpiceWorks software is also implemented for the management of the test bench and its creation..

Recomendaciones para el cliente

Instalaciones Eléctricas

  • No retirar las marquillas de las tomacorrientes reguladas y normales.
  • No sobrecargues los enchufes: No conectes demasiados aparatos a un mismo enchufe.
  • No tires del cable para desenchufar un aparato: Tira siempre del enchufe, no del cable.
  • No uses aparatos eléctricos con las manos mojadas: El agua y la electricidad son una combinación peligrosa.
  • Desconecta los aparatos electrónicos que no uses con frecuencia: Algunos aparatos electrónicos, como televisores y ordenadores, siguen consumiendo energía en modo de espera.

Instalaciones de Datos

  • Comprime los datos: Muchos navegadores y aplicaciones ofrecen opciones para comprimir los datos, lo que reduce la cantidad de datos que se descargan.
  • Mantén tu software actualizado: El software desactualizado puede contener vulnerabilidades que los piratas informáticos pueden aprovechar para acceder a tu dispositivo.
  • No retirar las marquillas de las tomas de datos

Conclusiones

El banco de pruebas cuenta con la instalación de tres puntos de red, tres puntos eléctricos normales y tres puntos eléctricos regulados, lo que demuestra una capacidad significativa para soportar múltiples dispositivos y configuraciones
